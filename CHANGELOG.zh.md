# 更新日志

[English](./CHANGELOG.md) | 中文

本项目的所有重要更改都将记录在此文件中。

## 0.3.0 - 2026-02-01

### 新功能
- 实现 Stage 详情页 Event Timeline (甘特图)，基于 ECharts 展示任务执行细节
- 支持未压缩的 Spark Standalone 模式事件日志
- 增强 Stage 摘要指标，实现与 Spark UI 的完全对齐

## 0.1.0 - 2026-02-01

### 新功能
- 启用任务表多列排序
- 增强作业/阶段分析及与 Spark History Server 的 UI 一致性
- 实现前端自动打包进 JAR

### 重构
- 模块化阶段详情并增强环境标签页
