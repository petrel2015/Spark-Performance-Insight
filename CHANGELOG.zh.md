# 更新日志

[English](./CHANGELOG.md) | 中文

本项目的所有重要更改都将记录在此文件中。

## 0.5.0 - 2026-02-01

### 新功能
- **Job 详情页**：新增专用 Job 详情视图，支持 Job 级别的 Executor 指标汇总及 Stage 列表过滤
- **Environment 页面重构**：按 Spark, Hadoop, JVM, System 等分类展示配置，支持独立搜索与折叠
- **Stage 列表 V2**：新增提交时间、耗时、输入输出、Shuffle 等核心列，优化表头命名
- **进度可视化**：在 Stage 列表中引入任务成功率进度条
- **调用栈信息**：Job 描述现在包含具体的调用位置信息（如 `csv at SparkETL.scala:20`）
- **Spark 版本徽章**：在导航栏和应用列表中展示 Spark 版本号
- **数据处理增强**：修复了 Task 成功状态识别逻辑，并统一将 0 值显示为 `-`
- **UI 一致性**：全站表格统一采用多列排序 UI 与 Hover 高亮效果

## 0.4.0 - 2026-02-01

### 新功能
- 新增 **Aggregated Metrics by Executor** 模块，支持按 Executor 聚合分析指标
- 新增 **指标选择器**，支持动态控制各组件中指标列/行的可见性
- 为 Stage 详情页的所有模块实现 **可折叠卡片** 功能
- 在页眉建立 Stage 与其所属 Job 的关联，支持点击跳转
- 优化记录数显示，同时支持精确计数与易读格式（如 `1,234 (1.2 K)`）
- 重构导航栏布局与样式，深度对标 Spark Web UI 体验
- 修复 Task 耗时与调度延迟计算逻辑，提升数据准确性

## 0.1.0 - 2026-02-01

### 新功能
- 启用任务表多列排序
- 增强作业/阶段分析及与 Spark History Server 的 UI 一致性
- 实现前端自动打包进 JAR

### 重构
- 模块化阶段详情并增强环境标签页
